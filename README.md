# Object-Oriented Programming Course

This is the support repository for the Object-Oriented Programming course taught at ASE bachelor Informatics Economics (taught in English). Materials are revised regularly and are meant to serve as aid for studying for the tests and exams.

> [!WARNING]
> Do not rely solely on this repo to study for the tests and exams without attending the laboratories and courses. In order to graduate this course ***it is mandatory to practice continuously and often***.
